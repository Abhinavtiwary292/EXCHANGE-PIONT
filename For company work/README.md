# Exchange Point - Premium Used Car Showroom Platform

Exchange Point is a highly professional, modern, and mobile-responsive multi-page web platform for a certified pre-owned car showroom. Inspired by leading platforms like Cars24 and Spinny, it features a premium dark/light navy aesthetic with vibrant amber accents and a clean glassmorphism design language.

## 🚀 Core Features

1. **Header & Navigation controls**:
   - Navigation links: Home, Buy Cars, How It Works.
   - Interactive **Select City** modal grid listing 24 popular Indian cities with landscape cropped landmark thumbnails, geolocation triggers, and real-time input filter queries.
   - Outlined **Call Us** action triggers a slide-out callback request drawer.

2. **Slide-Out Callback request drawer**:
   - Complete lead generation form featuring 10-digit phone validator, language selector (Default: Hindi), requirement textareas, operational disclaimers, and submit simulation overlays.

3. **Homepage Hero Section**:
   - Bold headlines ("Exchange Point - Drive In with Old, Drive Out with Bold.") and corrected "one roof" transparency taglines.
   - Dual Call-to-Action triggers redirecting users to inventory listings and valuation forms.

4. **Advanced Filters & Sidebar Controls**:
   - Budget range filter slider (₹0 Lakhs to ₹30 Lakhs).
   - Scrollable checkable list of 17 major automobile brands (Maruti Suzuki, Hyundai, Honda, Tata, Mahindra, Nissan, Volkswagen, Audi, BMW, Datsun, Renault, Jeep, Fiat, Citroen, Skoda, MG, and Kia).

5. **Dynamic Inventory Grid**:
   - Clean, structure-ready database loading interface designed to bind with external Google Sheets or REST APIs.
   - Renders modular placeholder elements for names, years, mileage specs, fuel/transmission badges, prices, real car image fallbacks, and certified trust tags.

6. **Interactive Valuation Calculator**:
   - Step-by-step profile analyzer calculating live trade-in values based on age, fuel parameters, mileage, and condition scores.

## 🛠️ Technology Stack
- **Structure**: Semantic HTML5
- **Styling**: Modern CSS3 (CSS Variables, Flexbox, CSS Grid)
- **Logic**: Vanilla Javascript (ES6 Modules)

## 📦 How to Run Locally
1. Clone this repository:
   ```bash
   git clone <your-repository-url>
   ```
2. Open `index.html` in any modern web browser, or use a development server extension (e.g., Live Server in VS Code).
